# Matematik Oyunu

Bu proje, kullanıcıların temel matematik becerilerini geliştirmelerine yardımcı olmak için geliştirilmiş bir oyun uygulamasıdır.

## Proje Kapsamı
- Toplama, çıkarma, çarpma ve bölme işlemleri
- Kullanıcıdan alınan cevapların doğruluğunu kontrol etme
- Puanlama sistemi ile ilerlemeyi takip etme

## Kullanılan Teknolojiler
- C#
- WinForms 

## Açıklama
Projede temel algoritmalar ve kontrol yapıları kullanılarak oyun mantığı geliştirilmiştir.  
Kullanıcı girişleri alınmış, doğruluk kontrolü yapılmış ve puanlama sistemi oluşturulmuştur.
